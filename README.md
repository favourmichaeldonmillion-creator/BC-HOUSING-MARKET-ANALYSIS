Project Title: BC Housing Market Analysis
Overview:
This project involves a comprehensive SQL-based analysis of the British Columbia housing market. The objective was to audit data integrity, establish temporal scope, and identify key market performance trends between 2021 and 2026.

Key Technical Skills Demonstrated:

Data Integrity Auditing: Used COUNT and IS NULL to ensure data reliability.

Temporal Context: Used MIN/MAX functions to define project scope.

Trend Analysis: Performed AVG aggregations and grouped data to analyze performance.

Advanced SQL: Implemented CASE logic for data categorization and subqueries for outlier detection.

Insights:
My analysis revealed that the "Total (house and land)" category consistently represents the highest market valuation segment. The analysis also highlights performance tiers, allowing stakeholders to identify "High-Value" market segments.
